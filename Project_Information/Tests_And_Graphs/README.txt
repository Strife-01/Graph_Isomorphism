README test.zip -- Test instances Implementation project Graph Isomorphism

A .gr file always contains a single graph and the only task is to compute the number of automorphisms (file name: ***AUT.gr).
A .grl file contains one or multiple graphs. The task is to find equivalence classes, if the file name is ***GI.grl; or to compute the number of automorphisms of each graph, if the file name is ***AUT.grl; or both, if the file name is ***GIAUT.grl.
